Calculator Tips

  1. It follows P.E.D.M.A.S Calculation Method
    • P - Parenthesis
    • E - Exponent
    • D - Division
    • M - Multiplication
    • A - Addition
    • S - Subtraction

  2. Write the operand or value before using any function or operator.

  3. For the sin, cos and tan functions, the angle values taken are measured in radians, NOT in degrees.

    4. Explanation:
    • The functions require values in radians. So, if we write 45 on the screen it will not be taken as 45° but as 45 rad.

    • Relation between radians and degrees :
      • π rad = 180°

    • Example : To obtain the value of tan(45°)
      • i) Convert from degree to radians

        Formula for conversion :
        • radian measure = (π / 180°) * degree measure


        Calculation :
        • radian measure (45°) = (π / 180°) * 45°
          radian measure (45°) = 0.7853981633975


      • ii) Press tan after receiving the screen value of the radian measure

        Calculation :
        • tan(0.7853981633975) = 1.0000000000001033

    That's how you use the trignometric functions in this Calculator.

  4. Special functions : ceil() and floor()

    • 4.1. ceil(x) - This function returns the smallest integer that is greater than or equal to x.

      • Example :
        • ceil(3.2) = 4
        • ceil(3) = 3
        • ceil(-3.2) = -3
        • ceil(-3) = -3

    • 4.2. floor(x) - This function returns the largest integer that is lesser than or equal to x.

      • Example :
        • floor(3.2) = 3
        • floor(3) = 3
        • floor(-3.2) = -4
        • floor(-3) = -3
